How Install This Theme

Easy Installation using your Hosting Panel or CPanel File Manager.

1.Download the last release of Theme after your Purchase.

2.Extract it, you will get 2 files, "LuxeTemplate.zip" and "luxe.zip".

3.Upload "LuxeTemplate.zip" into "modules" folder then extract it.

4.Upload "luxe.zip" into "public" folder then extract it.

That's it

Now You can select theme inside "site settings > setup".

Why the checkout keep old design?

Natively the themes are applied only on menu, but I've builded the add-on to restyling the checkout with the Luxe Theme UI.

Why some text is not translated?

Some text strings is not include on native code, you can add manually using the json file located on "/resources/lang/". For example, for Portuguese the file to add is "pt.json", for Spanish is "es.json". Write the exact text string (english) to traduce followed by exact translation in your language.

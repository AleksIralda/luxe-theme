Completely safe, This checkout theme not touch any code related to logic, behavior or process of payment. If you have any issue about the processing orders or something else this is not a issue from theme.

No, This add-on was specifically designed for luxe theme, this can't be used with ElegantTheme or DefaultTheme, Don't buy if you don't have Luxe Theme purchased.
